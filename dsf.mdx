## Overview

Personalize your in-game appearance on stew.gg by setting a custom rank color, tag, kill message, and death message to make yourself stand out.

<Warning>
  This feature is actively monitored and logged. Uploading inappropriate content will lead to **immediate removal** of your donor rank and **forfeiture of access**.
</Warning>

<Callout icon="key" iconType="regular" color="#658A95">
  Only players with the appropriate **donor rank** can use these commands.
</Callout>

## Customizing your rank color

You can change your rank color using the `/change rank color` command.

<CodeGroup>

```javascript Syntax
  /change rank color (color)
```

```javascript Example
  /change rank color aqua
```

</CodeGroup>

## Customizing your tag

You can set a custom tag above your head using the `/tags` command.

<CodeGroup>

```javascript Syntax
  /tags set (tag)
  /tags reset
```

```javascript Example
  /tags set [Chef]
```

</CodeGroup>

<br />

<Frame>
  ![Tag Preview](./images/customization/tag.png)
</Frame>

<br />

## Customizing your kill message

You can change your kill message using the `/custom kill message` command.

<CodeGroup>

```javascript Syntax
  /custom kill message (message)
```

```javascript Example
  /custom kill message {victim} got turned into stew by {killer}
```

</CodeGroup>

<br />

<Frame>
  ![Kill Message Preview](./images/customization/killmessage.png)

## Customizing your death message

You can change your death message using the `/custom death message` command.

<CodeGroup>

```javascript Syntax

```


## Customizing your eDate

You can send a relationship request to another player using the `/edate` command.

<Info>
  The receiving player will be prompted to accept as your **Girlfriend** or **Boyfriend** before the eDate is confirmed.
</Info>

<CodeGroup>

```javascript Syntax
  /edate (player)
  /breakup
```

```javascript Example
  /edate Steve
```

</CodeGroup>
